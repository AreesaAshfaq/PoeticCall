# Poetic Call

[Poetic Call](https://your-app-name.streamlit.app/) is an AI-powered poetry generation tool that creates beautiful poems based on your chosen subject and language. Using advanced language models, it generates creative and personalized poetry through a simple and user-friendly interface.

## Features

- **AI Poetry Generation:** Creates unique poems on any topic using AI language models.
- **Multi-Language Support:** Generate poetry in different languages by specifying your preferred language.
- **Simple User Interface:** Clean and interactive Streamlit interface for an easy poetry creation experience.
- **Creative Writing Assistant:** Helps users explore ideas, emotions, and creativity through AI-generated poems.

## How It Works

- **Enter Subject:** Provide the topic or theme of the poem (for example: Love, Nature, Friendship, or Spirituality).
- **Choose Language:** Select the language in which you want your poem to be generated.
- **Generate Poem:** Click the generate button to let AI create a personalized poem.
- **Read and Enjoy:** View your generated poem instantly and share your creative work.

## Technologies Used

- Python
- Streamlit
- Groq API
- Llama Language Model
